Sparkle Core Module

The core module contains basic Sparkle functionality required for all Sparkle applications, for example:

  • Audio & graphics (through AudioManager and GraphicsManager components
  • Event dispatching and handling (through the EventDispatcher component and Event classes)
  • 2D hitbox system for checking collision (Hitbox2D API)
etc.
